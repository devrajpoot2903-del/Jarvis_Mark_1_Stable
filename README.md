# 🤖 Jarvis Desktop Assistant (Mark 1.0)

A Python-based voice-controlled desktop assistant designed to automate daily desktop tasks using natural voice commands.

> ⚠️ This is the first functional version (Mark 1.0) and is currently under active development.

---

## 🚀 Features

- 🎤 Voice-controlled interaction
- 🗣️ Text-to-Speech responses
- 🎯 Wake Word Activation ("Jarvis")
- 🌐 Open websites using voice commands
- 💻 Launch desktop applications
- 📂 Open File Explorer
- ⚙️ Perform system operations
  - Sleep
  - Shutdown
  - Restart
- ⏱️ Command timeout for better workflow
- 🎧 Continuous listening mode
- 🖥️ Personalized desktop automation

---

## 🛠️ Tech Stack

- Python 3.12

---

## 📌 How It Works

1. Jarvis continuously listens for the wake word.
2. Say **"Jarvis"** to activate the assistant.
3. Jarvis responds with **"Yes Sir"**.
4. Speak your command.
5. Jarvis executes the requested task.
6. Returns to listening mode.
